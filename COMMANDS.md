# Create a Angular shell
```yaml
npx create-nx-workspace name     
```
# Add Angular project  
```yaml
nx g @nrwl/angular:app ng-mfe 
```
# Add React project
```yaml
yarn add @nrwl/react       
nx g @nrwl/react:app name
```
# Install all dependencies
```yaml
yarn
```
# Run all remote projects
```yaml
yarn run start:all
```
